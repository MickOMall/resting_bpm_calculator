Use instructions :
1. Download all files in the repo to a folder of choice.
2. Inside this folder have a folder named "resting_bpm_app" in the main folder.
3. Inside "resting_bpm_app" make one more folder that is named "model"
4. The model.ipynb and the data.csv should be in your folder of choice, and the app.py should be in the resting_bpm_app folder.
5. Run the code in the model.ipynb file to train and download the forest regression model.
6. Then go to the "resting_bpm_app" directory in your terminal and run "streamlit run app.py"
